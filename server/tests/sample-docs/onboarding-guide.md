# Employee Onboarding Guide

## Purpose

This guide helps new team members get set up during their first week.

## Day 1 Checklist

- Create company email account
- Join project communication channels
- Install required tools (`git`, `docker`, `uv`, `node`)
- Request access to staging and analytics dashboards

## First Week Goals

1. Run the backend service locally.
2. Run the admin UI and verify login.
3. Upload one sample file and run a search query.
4. Document one issue or confusion in the team notes.
